📝 Assignment

Users API documentation
Part 1
  1. Retreive all users from the server and display in table
  2. Add search input above the table and implement client side filtering
  3. Add activate 🟢 / deactivate 🔴 icon on every row based on the "activated" property from user object. (You can use these emoji icons)
  4. Pressing on the above mentioned icon should toggle user activation state
  5. Add delete button on every row
  6. Pressing on delete ❌ button should delete the user
Part 2
  • Add the "Add new user" button above the table
  • Pressing on the "Add new user" button should open the user form
  • Using this form the user has to be able to add new user
Part 3
  • On every row add the edit ✏️ button
  • Pressing on the edit button should open the same user form with the prefilled values
  • Using this form the user has to be able to update the existing user
📢️ Notes
  • The data is stored in memory, so restarting the server will cause loosing newly added data
  • You can use React, Angular or just vanilla JavaScript if you want
  • You can use any 3rd party library you want. Including UI component libraries
  • Try to keep you code clean and DRY
  • Feel free to use Google as well
